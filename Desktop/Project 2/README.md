A simple, responsive Employee Management System built with HTML, CSS, and JavaScript.
This app allows users to register employees, view them in a table, search by name, and delete records.

Some features include:
    - Register new employees
    - Display employees in a table
    - Responsive layout with media queries
    - Search employees by name

Technologies Used:
    - HTML
    - CSS
    - JavaScript
